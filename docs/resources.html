Resources

  • Mozilla Developer Network - Great resource for learning and refreshing your knowledge of front-end development technologies. Whether you are a true beginner or have moderate experience, MDN is always a great resource to have!
  • W3Schools - A vast amount of information and tutorials that you can read and try for free of varying front-end development technologies. W3Schools offers the ability to test your skills and earn certifications at a cost, but still offers a tremendous amount of free information.

Tutorials & Courses

  • The Complete JavaScript Course by Jonas Schmedtmann - This course offered by Udemy will cost $10 but does a great job of explaining the basics of JS and how it really works at the core. If you are looking for some help with DOM traversal this is a great resource. There are also a list of wonderful resources he has compiled on his personal webpage, CodingHeroes.io
  • SoloLearn - Great little resource if you are looking for help with core programming fundamentals. SoloLearn offers fundamental tutorials on plenty of languages!
  • Udemy - A vast library of learning resources for all kinds of topics. This link will take you to the development side, which offers thousands of courses for every programming language and topic you could imagine.

Books

  • Eloquent JS - You can find a link to the paperback version as well here, or just utilize the Electronic versions. Some taught Eloquent JS to be the be-all-end-all reference for Vanilla JS
  • You Don't Know JS - A series of learning fundamental JavaScript published by Kyle Simpson. You can find the electronic versions here as well as links to the paperback versions as well.
